# analysis

Systematic analysis of codebases, data, and technical decisions to produce actionable findings and recommendations.

## Usage

Load this skill when you want an AI agent to analyze code, data, or technical decisions and produce a structured report.

**Example prompts:**
- "Analyze the trade-offs between REST and gRPC for our use case"
- "Analyze the dependency graph of this project for security risks"
- "Assess the technical debt in the authentication module"

## What It Does

1. Clarifies the question being answered
2. Gathers evidence from code, data, documentation, and other sources
3. Identifies patterns, anomalies, and root causes
4. Evaluates trade-offs with explicit criteria
5. Produces prioritized findings with specific citations
6. Recommends actions ranked by impact and effort

## Compatible Engines

- claude-code
- gemini-cli
