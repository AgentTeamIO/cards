# code-generation

Generate production-ready code from natural language requirements, matching existing codebase patterns and conventions.

## Usage

Load this skill when you want an AI agent to generate new code from specifications, feature requests, or task descriptions.

**Example prompts:**
- "Generate a REST API handler for user authentication"
- "Create a rate limiter using the token bucket algorithm"
- "Implement a file upload service with size validation"

## What It Does

1. Reads and interprets the requirements
2. Studies the existing codebase for patterns and conventions
3. Plans the implementation structure
4. Generates idiomatic, production-ready code
5. Produces corresponding tests

## Compatible Engines

- claude-code
- gemini-cli
- codex
