# documentation

Technical documentation skill that generates clear, accurate, and maintainable docs from code analysis.

## Usage

Load this skill when an AI agent needs to write or update technical documentation.

**Example prompts:**
- "Document the public API for this REST service"
- "Write an architecture overview for this project"
- "Create a README for this module"

## What It Does

1. Determines the doc type and audience (API reference, README, architecture overview, runbook)
2. Analyzes the code to extract the public API surface, data flow, and configuration
3. Writes structured documentation with examples
4. Verifies accuracy against the source code

## Compatible Engines

- claude-code
- gemini-cli
